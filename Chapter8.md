# 磁盘存储器的管理
## 外存的组织方式
### 连续组织方式
- 优点
    - 顺序访问容易
    - 顺序访问速度快
- 缺点
    - 要求为一个文件分配连续的存储空间
    - 必须事先知道文件长度
    - 不能灵活地删除和插入记录
    - 对于那些动态增长的文件很难分配空间

### 链接组织方式
- 优点
    - 消除了磁盘的外部碎片，提高了外村利用率
    - 对插入、删除和修改记录都非常容易
    - 能适应文件的动态增长，无需事先知道文件大小

- 隐式链接与显式链接
- 簇（Cluster）

- 文件分配表FAT
    - 卷（分区）
    - FAT12、FAT16、FAT32
- NTFS(New Technology File System)
    - 逻辑簇号LCN、虚拟簇号VCN
    - 主控文件表MFT
